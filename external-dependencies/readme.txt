threedi_leak_detector depends on scipy. To install it for QGIS, follow these instructructions:

	https://lutraconsulting.co.uk/blog/2016/03/02/installing-third-party-python-modules-in-qgis-windows/
	
Alternatively, copy the folders in 'external-dependencies' to {your-qgis-profile-folder}\python
You can open your QGIS profile directory from QGIS: Settings > User Profiles > Open Active Profile Folder
